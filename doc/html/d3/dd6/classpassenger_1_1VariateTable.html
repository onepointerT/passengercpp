Detailed Description

template<typename ColT, typename RowT, typename CellT, typename num = num_non_default>
class passenger::VariateTable< ColT, RowT, CellT, num >

A Table< CellT, num > that is able to save objects for the row's and column's heading.

Template Parameters
ColTThe type to be saved for the column's heading, no pointer allowed here, the pointer is added by-design
RowTThe type to be saved for the row's heading, no pointer allowed here, the pointer is added by-design
CellTThe type to be saved for the cells, no pointer allowed here, the pointer is added by-design
numThe number type to enumerate the table with
