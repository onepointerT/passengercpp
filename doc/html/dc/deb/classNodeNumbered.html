Detailed Description

template<class GraphPassengerT = GraphPassengerNumber<>, typename num = num_default>
class NodeNumbered< GraphPassengerT, num >

A numbered and controlled node, inheriting Node< GraphPassengerT > and PassengerNumber< num >.

Note
This is the last class without an edge as template parameter. Please have this in mind because of cyclic template parameters. NOTE: Base classes also work as a template parameter.
Template Parameters
