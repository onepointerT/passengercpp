Detailed Description

template<typename CellT, typename num = num_non_default>
class passenger::Table< CellT, num >

A classic table with indices on rows and columns, that can save an object in its cells.

Template Parameters
CellTThe type of object to save in the cells, no pointer allowed here, the pointer is added by-design
numThe type of numeration for the rows and columns
